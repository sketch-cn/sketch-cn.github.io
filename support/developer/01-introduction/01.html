Plugin Basics

What is a plugin?

In its most basic form, a plugin is just a text file with code that tells Sketch what to do. More complex plugins may consist of a collection of files and folders, and some can even share code by using a common library.

Sketch Plugins use the .sketchplugin extension, and are written in JSTalk (we'll talk a bit more about JSTalk in the Getting Started section.)

Installation / Where are plugins located?

Basic plugins are installed by double-clicking the .sketchplugin file. This copies it to the Plugins folder:

  • ~/Library/Application Support/com.bohemiancoding.sketch3/Plugins if you bought your copy directly on our site or
  • ~/Library/Containers/com.bohemiancoding.sketch3/Data/Library/Application Support/sketch/Plugins if you are using the App Store version.

You can access the Plugins folder easily by using the Reveal Plugins Folder… menu option from Sketch:

Removal

To remove a Plugin, just delete it from the Plugins folder. Select the Plugins › Reveal Plugins Folder… menu option and drag the Plugin you want to remove to the Trash. It will be removed from the Plugins menu immediately (you don't need to close and reopen Sketch :)
