Sharing Sketch Files

I've sent my friend my .sketch file and it arrived empty, why is this happening?

Sketch files are contained folders, folders that are not supported by many file sharing technologies, the most popular being e-mail and backup services like dropbox, google drive.

In order to safely share a .sketch file with co-workers and friends we recommend you to always .zip it before sharing, this way all the folder structure is kept and sent, no data loss occurs.

Can I backup my files to Dropbox or Google drive?

Yes, but we recommend that you wait for the file to be fully synced before opening it on another computer, in case you don't, different computers will sync the same elements to the same file and can potentially damage it.
