Getting Started

If you've decided to take the toolmaker journey, there are some things you definitely need to know before you get started.

Syntax

Sketch Plugins are made possible by JSTalk, a bridge that lets you use Objective-C/Cocoa code from an external script written in JavaScript. The bridge takes care of the translation between JavaScript and Cocoa, so you can concentrate on the important parts (namely, making Sketch do awesome stuff).

From JSTalk's README:

JSTalk is built on top of Apple's JavaScriptCore, the same JavaScript engine that powers Safari. So when you write in JSTalk, you are really writing JavaScript.

JSTalk also includes a bridge which lets you access Apple's Cocoa frameworks from JavaScript. This means you have a ton wonderful classes and functions you can use in addition to the standard JavaScript library.

You can use two different styles when writing your scripts: dot notation and bracket notation.

If you have spent some time writing JavaScript code, you already know about the dot notation:

var l = a.length()

Here, a dot is used to say: "Call the 'length' method on the 'a' object".

However, if you come from a Cocoa background, you may be happy to learn that you can also use a bracket notation, very similar to that in Objective-C:

var l = [a length]

You can even mix and match dot and bracket notation:

var l = [a length].toString()

However, it's probably a good idea to stick with just one style.

Preamble / Default Variables

All Plugins in Sketch have access to the following default variables:

  • doc, an MSDocument object which is a reference to the current document,
  • selection, an NSArray of the currently selected layer(s), and
  • scriptPath, an NSString containing the path to the current Plugin.

Accessing Cocoa

You can access all Cocoa APIs from JSTalk. For example, if you want to open a File Picker panel, you can use the NSOpenPanel class:

var openPanel = [NSOpenPanel openPanel]
[openPanel setCanChooseDirectories:NO]
[openPanel setCanChooseFiles:YES]
[openPanel setCanCreateDirectories:NO]
[openPanel setDirectoryURL:[NSURL fileURLWithPath:"~/Documents/"]]

[openPanel setTitle:@"Choose a file"]
[openPanel setPrompt:@"Choose"]
[openPanel runModal]

If you need more information about Cocoa, check the Resources section.

Other minor stuff you need to know:

  • Semicolons are optional in JSTalk.
  • You can evaluate falseness by using false or NO
  • You can evaluate truthiness by using true or YES
  • You can evaluate nullness by using null or NIL
